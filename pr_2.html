HTML Text Formatting Tags

Tag Description Example
<b> Bold text. <b>This is bold text.</b>
<i> Italic text. <i>This is italic text.</i>
<u> Underlined text. <u>This is underlined text.</u>
<strong> Important text (bold). <strong>This is important text.</strong>
<em> Emphasized text (italic). <em>This is emphasized text.</em>
<mark> Highlighted text. <mark>This text is highlighted.</mark>
<small> Smaller text. <small>This is small text.</small>
<sub> Subscript text. <sub>This is subscript text.</sub>
<sup> Superscript text. <sup>This is superscript text.</sup>
<del> Deleted text (strikethrough). <del>This text is deleted.</del>
<ins> Inserted text (underline). <ins>This text is inserted.</ins>
<blockquote> Block of quoted text. <blockquote>This is a quote.</blockquote>
<code> Inline code. <code>console.log("Hello, World!");</code>
<pre> Preformatted text. <pre>This is preformatted text.</pre>
